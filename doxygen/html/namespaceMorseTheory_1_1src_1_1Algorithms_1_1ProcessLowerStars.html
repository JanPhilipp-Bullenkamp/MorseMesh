Extracts the lower star of a vertex.

Contains the vertex and all edges and faces with lower function values than the vertex.

Parameters
vertexThe vertex we want to get the lower star of.
edges_dictDictionary containing all edges.
faces_dictDictionary containing all faces.
Returns
lower_star A dictionary containing the keys 'vertex', 'edges' and 'faces' which each contain key value pairs of index and edge/face (Simplex class objects). The edges and faces are sorted by function values.

The function described in Robins et al.

2011, that returns a discrete gradient vector field.

Loops over all vertices, calculates their lower stars and pairs up as many edge-vertex (V12) or face-edge (V23) pairs as possible. Simplices that cannot be paired will become critical simplices and are stored in C.

Parameters
vertices_dictThe dictionary containing all vertices.
edges_dictThe dictionary containing all edges.
faces_dictThe dictionary containing all faces.
CThe dictionary that will store all critical vertices, edges and faces.
V12The dictionary that will store pairings between edges and vertices.
V23The dictionary that will store pairings between faces and edges.
Returns
Updated C, V12 and V23.
