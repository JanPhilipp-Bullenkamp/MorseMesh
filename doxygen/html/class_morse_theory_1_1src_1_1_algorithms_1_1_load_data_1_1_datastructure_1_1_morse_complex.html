Detailed Description

A Morse Complex, that stores a reduced skeleton of the original mesh/simplicial complex and has the same topology.

A Morse Complex consists of critical vertices, edges and faces, representing minima, saddles and maxima of the assigned scalar Morse function on the mesh. These critical simplices are connected via separatrices, that give information on how the gradient "flows" from the maxima to the saddles and from the saddles to the minima. This gives a topological skelton of the mesh which can be noise reduced using a parameter called persistence. The topologicallly relveant Betti numbers can be stored as well.
