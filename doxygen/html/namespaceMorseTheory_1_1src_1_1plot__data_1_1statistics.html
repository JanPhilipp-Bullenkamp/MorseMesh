Creates statistics of function values on all critical vertices, edges and faces separately and allows to optionally plot and save the histograms as well.

the histograms will be plotted adding 'critV', 'critE' and 'critF' to the filepath.

Parameters
MSComplexThe Morse Complex we want to have the function value statistics of (will use CritV, CritE and CritF).
nb_bins(Optional) Integer. The number of bins for the histogram. Default is 15.
log(Optional) Bool. Use logarithmic scale for the counts /y-axis in the histogram. Default is False.
save(Optional) Bool. Whether to save the histogram as a file. Default is False.
filepath(Optional) The filepath to use if the histogram should be saved. Default is 'histogram'.
show(Optional) Bool. Whether to plot the histogram or not. Default is True.
Returns
stat A dictionary containing the keys 'V', 'E' and 'F' each containing dictionaries with keys 'mean', 'std' and 'fun_vals' containing the mean, the standard deviation and a list of the function values for the critical vertices, edges or faces respectively.
