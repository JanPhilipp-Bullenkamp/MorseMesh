Morse Theory Framework

Description

A framework for using discrete Morse Theory on 3D triangluar meshes. Can be used for segmentation when loading in a suitable scalar function (e.g. curvature values on each vertex).

Contains

A morse class that can load a 3D mesh (ply files only at the moment) and perform various Morse Theory computations, as well as a segmentation method and various plotting functions.

Libraries to install

  • plyfile library (https://github.com/dranjan/python-plyfile)
    • Required for reading ply file
