Uses double threshold to get strong and weak edges and adds weak edges to strong edges if they are adjacent to a strong edge.

Parameters
maxRedCompA maximally reduced Morse Complex (needed for the Separatrices).
thresh_highThe high threshold for edges that will definitely be added.
thresh_lowThe weak threshold for edges that will be added of they are adjacent to a strong edge.
vert_dictDictionary containing all vertices.
edge_dictDictionary containing all edges.
face_dictDictionary containing all faces.
Returns
strong_edge The double thresholded edges stored as a single set of vertex indices.
