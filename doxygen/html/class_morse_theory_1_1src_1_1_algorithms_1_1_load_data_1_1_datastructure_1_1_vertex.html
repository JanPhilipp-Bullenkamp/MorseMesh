Detailed Description

Vertex class used for normal vertices and their properties.

Stores the following properties of a vertex:

  • x,y and z coordinate
  • a quality value
  • a function value
  • an index
  • two angles theta and phi (for normals e.g.)
  • a star dictionary with keys "F", which can store the adjacent face-indices in a list, and "E", which can store the adjacent edge-indices in a list
  • a set .neighbors to store the neighbors of the vertex
