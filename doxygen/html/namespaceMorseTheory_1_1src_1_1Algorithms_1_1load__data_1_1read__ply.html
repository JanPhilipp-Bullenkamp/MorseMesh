Reads a ply file and writes the simplicial complex into vertices, edges and faces dictionaries.

This function reads the given ply file and fills the given vertices dictionary, edges dictionary and faces dictionary. It uses the values given at the quality index position as Morse function values and makes sure that they are all unique so that we can use it for discrete Morse theory. Also allows to flip the function values by multiplying with -1 and thereby switching minima and maxima.

Parameters
filenameThe ply file to be read.
quality_indexThe position of the quality you want to read in as the Morse function. Might vary depending on the ply file and application.
vertices_dictThe vertices dictionary to be filled with vertices.
edges_dictThe edges dictionary to be filled with edges.
faces_dictThe faces dictionary to be filled with faces.
invertedOptional boolean: whether the Morse function values (quality values) should be multiplied with -1. Doing this flips maxima and minima, default is False
Returns
Despite filling the dictionaries, returns the minimum and maximum function value as min, max.
