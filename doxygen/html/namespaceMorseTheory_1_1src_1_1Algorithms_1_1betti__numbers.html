def MorseTheory.src.Algorithms.betti_numbers.betti_via_pair_cells (   MorseComplex)

The wrapper for the PairCells algorithm described by Zomorodian.

Calculates Betti numbers by pairing up all critical simplices of the Morse Complex; leaving some unpaired. The unpaired cells give us the betti numbers while the paired cells can be used to obtain a persistence diagram.

Parameters
MorseComplexA Morse Complex class object that we want to calculate the betti numbers from.
Returns
betti, partner0, partner1, partner2 The betti numbers in a 3 long array, and the paired cells for each dimension in dictionaries.
