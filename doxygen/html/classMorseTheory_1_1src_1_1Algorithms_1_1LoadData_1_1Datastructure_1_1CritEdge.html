Detailed Description

CritEdge class used for critical edges.

Stores the indices, function values and edge-index of a critical edge. Further contains a list with minima and a list with maxima (critical vertices and faces) connected to this critical edge (.connected_minima and .connected_maxima). Also contains a paths dictionary (.paths) (?needed for MorseComplex computations?).
