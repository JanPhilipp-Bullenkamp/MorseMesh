Detailed Description

A Morse Complex, that stores a reduced skeleton of the original mesh/simplicial complex and has the same topology.

Also stores MorseCells and various Segmentations for this persistence level.

A Morse Complex consists of critical vertices, edges and faces, representing minima, saddles and maxima of the assigned scalar Morse function on the mesh. These critical simplices are connected via separatrices, that give information on how the gradient "flows" from the maxima to the saddles and from the saddles to the minima. This gives a topological skelton of the mesh which can be noise reduced using a parameter called persistence. The topologicallly relveant Betti numbers can be stored as well. Since the skeleton provides us with enclosed areas on a mesh as well, each Morse Complex can also contain MorseCells, which give a segmentation in between the skeleton of the mathematical complex, as well as different segmentations that started from the MorseCells of the MorseComplex and got reduced based on defined ridges.

Creates a segmentation from this MorseComplex with the given double edge threshold and the merging threshold.

Creates a copy of the Morse cells at this persistence level and than segments based on the given parameters. Segemntation stops when either no more cells can be merged due to no more adjacent cells having a weight below the merge threshold, or reaching the optional minimum number of labels (which is defaulted to 3).

Parameters
salient_edge_pointsThe salient edge points that have been calcualted from the double threshold
thresh_largeThe larger threshold of the double threshold that was used to get the edge points.
thresh_smallThe smaller threshold of the double threshold that was used to get the edge points.
merge_thresholdThe threshold that determines when to stop merging cells. Weights in the neighborhood graph of the Morse cells will be above this threshold.
minimum_labelsThe minimum number of labels we want to keep. Default is set to 3
