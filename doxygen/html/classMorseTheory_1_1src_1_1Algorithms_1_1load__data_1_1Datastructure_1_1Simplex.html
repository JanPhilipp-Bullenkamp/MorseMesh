Detailed Description

Simplex class used for normal edges and faces.

Stores an edge or face as a set of two/ three indices (representing two/ three vertices). Each simplex has an own index (edge-index or face-index).

Can set the function value of the simplex by taking the function values of the vertices and storing them in a list, sorted such that the higher function value is at the first position.

Checks if a given other simplex is a face of this simplex.

Another simplex is a face of this simplex, if its vertices are a real subset of this simplex vertices and there is only one vertex not contained. (e.g. an edge with 2 indices can be the face of a triangle with 3 indices.

Parameters
other_simplexAnother Simplex object that we want to know of whether it is a face of this simplex.
Returns
Bool. True if the other simplex is a face of this simplex, False otherwise.
